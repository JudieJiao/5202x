2. L4 Multiple Comparisons 多重比较

2.1 为什么要多重比较?

对 $m$ 个检验,每个用 $\alpha$ → Family-Wise Error Rate (FWER) $\approx 1-(1-\alpha)^m$。例:$\alpha=0.05, m=100 \Rightarrow$ 至少一个错误拒绝的概率 $=1-0.95^{100}\approx\mathbf{0.9941}$(近乎必错)。

FWER 定义:$P\!\left(\bigcup_{j\in C}\{T_j\ge c_j\}\right)$。目标:找 critical value 把此概率控制在 $\alpha$。

2.2 通用检验统计量

★ Test statistic for any contrast $$T_a = \frac{a^\top\hat\theta}{\sqrt{a^\top\hat\Sigma_\theta\, a}}$$ 其中 $\hat\theta$ 是估计向量(如 $\hat\mu$ 或 $\hat\beta$),$\hat\Sigma_\theta$ 是其 var-cov 矩阵。

2.3 四大准则 Four Criteria ★必考

准则适用场景Critical value (two-sided)
Scheffé 任何 contrast,含数据驱动 / general exploration;$q$ 个 group means $c_\alpha=\sqrt{m_1\,F_{m_1,m_2}(\alpha)}$;主效应场合 $m_1=a-1,\,m_2=n-a$: $\sqrt{(a-1)F_{a-1,n-a}(\alpha)}$
Tukey (HSD) 全部 pairwise,要求 equal n $c_\alpha=q_{a,\,n_{err}}(\alpha)/\sqrt{2}$(Studentized range quantile)
Dunnett 所有 treatment vs 1 个 control 查 Dunnett 表 $d_{a,\nu,\alpha}$;unequal $n$ 时乘 $m_j=1+0.07(1-n_j/n_1)$
Bonferroni 有限个 pre-specified contrasts/tests $t_{n-p}(\alpha/(2k))$;simultaneous p-value = $k\times$ individual p
★ 决策规则 Decision Rule $$\text{Reject } H_0 \iff |T_a| > c_\alpha,\qquad \text{CI: } a^\top\hat\theta \pm c_\alpha\cdot \sqrt{a^\top\hat\Sigma_\theta a}$$

2.4 Studentized Range 公式

对 $a$ 个 groups,$n_{err}=n-a$:

$q_{a,n_{err}}=\dfrac{\sqrt{n}\,(\max \bar Y_i - \min\bar Y_i)}{\hat\sigma} = \sqrt{2}\,\max|T_{ij}|$

2.5 Scheffé 单侧 one-sided 调整(Supplement L4)

将 $\alpha$ 替换为 $2\alpha$:$c_{\text{one-sided}}=\sqrt{(a-1)F_{a-1,\,n_{err}}(2\alpha)}$。

2.6 关键关系:Overall F vs Contrasts

易错点: Overall ANOVA F 可以显著,而没有一个 inspected contrast 显著(反之亦然)。overall F 等价于"Scheffé 下至少一个 contrast 显著"的 dual。

2.7 如何选准则?(高频判断题)

口诀:
  1. 仅限 pairwise + equal n → Tukey(通常最紧)
  2. 所有 possible contrasts(含事后探索) → Scheffé
  3. vs control → Dunnett
  4. $k$ 个预先指定 + $k$ 小 → 比较 Bonferroni 与 Tukey/Scheffé,c 小者胜
  5. directional(one-sided)且非 pairwise → Tukey 不能直接用

2.8 R 实现

fit <- aov(y ~ group, data=d)
TukeyHSD(fit)                              # Tukey HSD
pairwise.t.test(y, group, p.adjust="bonferroni")
library(multcomp); summary(glht(fit, linfct=mcp(group="Dunnett")))
# Scheffé: 手动构造 C matrix
C <- rbind(c(1,-1,0,0))                    # contrast matrix
bhat <- coef(fit); VC <- vcov(fit)
est  <- C %*% bhat; se <- sqrt(diag(C %*% VC %*% t(C)))
t_stat <- est/se
# 比较 |t_stat| 与 Scheffé critical

2.9 例题(Past Paper Q1 类型)

3 brands,lm(y~brand) 给出 brand2 est=−15.8 (SE=2.498), brand3 est=5.2 (SE=2.498), $n-p=12$。预期 $\mu_2<\mu_1<\mu_3$(2 个 one-sided,$k=2$)。

  • $t_1^*=-6.325$;  $t_2^*=2.082$
  • Bonferroni one-sided critical $=t_{12}(0.05/2)=\mathbf{2.1788}$
  • $|t_1^*|>2.1788$ → reject($\mu_2<\mu_1$ 成立);$|t_2^*|<2.1788$ → not reject

3. L5 ANCOVA & 非线性预测项 注意

⚠️ 注意:L5 不是 general linear test 专题!它讲的是两类模型扩展:
(i) ANCOVA(协方差分析) — 同时含 factor 和 continuous covariate
(ii) 非线性预测项 — 多项式 / log-log / inverse / piecewise linear

3.1 ANCOVA 基本模型

$$y_i=\beta_0+\sum_{j=2}^{k}\alpha_j u_{ij}+\gamma\, x_i+\epsilon_i$$ $u_{ij}$ = factor 的 dummy;$x_i$ = covariate(协变量);$\gamma$ = 协变量斜率(假定各组相同)。

3.2 Comparison-of-regression-functions 模型(各组斜率不同)

$$Y=\beta_0+\sum_j\alpha_j U_j+\beta X+\sum_j\xi_j U_j X+\epsilon$$
  • 不同 intercept: $\beta_0+\alpha_j$
  • 不同 slope: $\beta+\xi_j$
  • 检验"平行性"$H_0:\xi_j=0\;\forall j$

3.3 "When $x$ is held at its mean" 常用技巧

令 $z=x-\bar x$,模型 $y=\beta_0+\beta_2 T_2+\beta_3 T_3+\alpha z+\gamma_1 T_2 z+\gamma_2 T_3 z$。
$z=0$ 时:group means = Intercept + treatment effects。Contrast $\mu_2-\mu_1=\beta_2$,$\mu_3-\mu_1=\beta_3$。

3.4 非线性预测项 Non-linear predictor terms

类型模型形式注意
Polynomial 多项式$y_i=\beta_0+\beta_1(x_i-\bar x)+\beta_2(x_i-\bar x)^2+\cdots$中心化减少共线性;但最高阶系数与是否中心化无关
Inverse 反函数$y_i=\beta_0+\beta_1(1/x_i)+\epsilon_i$适用渐近水平关系
Log-log$\log y=\beta_0+\beta_1\log x+\epsilon$弹性解释;$\beta_1$ = $x$ 每增 1% 导致 $y$ 变 $\beta_1\%$
Piecewise linear 分段线性$y_i=\beta_0+\beta_1 x_i+\alpha_1(x_i-c)^+ +\epsilon_i$$x<c$ 时斜率 $\beta_1$,$x\ge c$ 时 $\beta_1+\alpha_1$;检验 $H_0:\alpha_1=0$
重点:"linear" in LRM 指"linear in parameters"。$y=\beta_0+\beta_1 x^2$ 仍然是线性模型;$y=\beta_0\,x^{\beta_1}$ 不是。

3.5 R 实现

lm(y ~ x + type)                     # ANCOVA (same slope)
lm(y ~ x * type)                     # Different slopes (interaction)
lm(y ~ x + I(x^2) + I(x^3))          # Polynomial
lm(y ~ I(1/x))                       # Inverse
lm(log(y) ~ log(x1) + log(x2))       # Log-log
z = (x - 250) * (x >= 250)           # Piecewise at c=250
lm(y ~ x + z)

3.6 anova() 顺序问题

顺序依赖! anova(lm(y~A*B)) 输出的是 Type I (sequential) SS,A 先进、B 后进,交互最后。若想要"加入 B 后 A 的独立贡献",需要反序或用 drop1()。

4. L6 Model Selection 模型选择(含正则化)

4.1 Bias-Variance 分解

$$\text{MSE} = \text{Var} + \text{bias}^2$$ SPSE (Sum of Predicted Squared Error): $$\text{SPSE} = n\sigma^2 + |M|\sigma^2 + \sum_i(\mu_i-\mu_{iM})^2$$

4.2 选择准则 Criteria ★

Criterion公式方向
$R^2$$SSR/SST$越大越好 但会一直随 p 增
Adjusted $R^2$$R_a^2 = R^2 - \dfrac{p}{n-p-1}(1-R^2)$越大越好
Mallow's $C_p$$C_p=\dfrac{SSE_M}{\hat\sigma^2_{\text{full}}}-n+2|M|$越小越好;理想 $E[C_p]\approx|M|$
AIC$-2\log L+2j_M$;LRM: $n\log\hat\sigma^2_M+2j_M+C$越小越好
BIC$-2\log L+j_M\log n$越小越好(更严,倾向 sparse)
EBIC$\text{BIC}+2\gamma\log\binom{p}{j_M}$,  $\gamma=1-\dfrac{\log n}{2\log p}$高维场合
$k$-fold CV$\tfrac{1}{n}\sum(y_i-x_i^\top\hat\beta^{(-\text{fold}(i))})^2$越小越好
LOO CV$\tfrac{1}{n}\sum(y_i-x_i^\top\hat\beta^{(-i)})^2$越小越好

4.3 三种逐步算法 Stepwise

  • Forward: null → 每步加入使 AIC 最小的变量
  • Backward: full → 每步删除最不显著的
  • Stepwise (both): 每步先考虑加再考虑删

4.4 读 stepAIC 输出 ★高频

Step: AIC=276.73
y ~ x3
        Df     SSR     SSE
+ x2     1   52502    2210      ← SSR 最大 → 候选
+ x4     1   10639   44073
<none>                  54712   ← 当前模型 SSE
+ x1     1    2887   51825
  • SSR = $SSE_{\text{old}}-SSE_{\text{new}}$(减少量)
  • 候选 = SSR 最大行
  • 加入条件:新 AIC $<$ <none> AIC
  • 近似:$\Delta\text{AIC}\approx n\ln(SSE_{\text{new}}/SSE_{\text{old}})+2$

4.5 正则化 / Shrinkage(L6 已引入,L8 继续用)★

★ 惩罚最小二乘 $$\min_\beta \; \tfrac{1}{n}\|y-X\beta\|^2 + p_\lambda(\beta)$$
方法惩罚变量选择特性
Ridge$\lambda\sum\beta_j^2$否凸;所有 coef 缩小;允许 $p>n$
LASSO$\lambda\sum|\beta_j|$是 (sparse)凸;$\ell_1$;同时选变量和估计
Adaptive LASSO$\lambda\sum w_j|\beta_j|,\; w_j=|\hat\beta_j|^{-1}$是凸;oracle 性质
SCADSmoothly Clipped AD是非凸;大系数无惩罚
MCPMinimax Concave是非凸;$|\beta|>\gamma\lambda$ 时惩罚为 0

4.6 $\lambda$ 的选择

  • Prediction 为主 → CV / GCV 选 $\lambda_{\min}$(CV error 最小)
  • Variable identification 为主 → BIC / EBIC
  • 保守(更少假阳性) → lambda.1se(1 SE rule)

4.7 R 实现

library(MASS)
stepAIC(lm(y~1, data=d), scope=list(upper=lm(y~., data=d)), direction="forward")
stepAIC(lm(y~., data=d), direction="backward")
stepAIC(lm(y~1, data=d), scope=list(upper=lm(y~., data=d)), direction="both")
AIC(fit, k=2);  AIC(fit, k=log(n))     # BIC via k=log(n)

library(boot); cv.glm(d, glmfit, K=10)$delta   # k-fold CV

library(glmnet)

cv$lambda.min;  cv$lambda.1se
coef(cv, s="lambda.min")
predict(cv, newx=X_test, s="lambda.min")

5. L7 Regression Diagnostics 回归诊断 重点

5.1 Hat Matrix & Leverage

$$H = X(X^\top X)^{-1}X^\top,\quad h_{ii}=x_i^\top(X^\top X)^{-1}x_i,\quad \sum_i h_{ii}=p$$ $$\mathrm{Var}(e_i)=\sigma^2(1-h_{ii})$$
  • $h_{ii}\in[0,1]$,平均 $=p/n$
  • 阈值:$h_{ii}>2p/n$ → high leverage
  • 仅依赖 X,不依赖 y

5.2 三类残差 Three Residuals ★必背

★ 核心公式 — 3 Residuals $$\text{Raw:}\; e_i=y_i-\hat y_i$$ $$\text{Studentized (internal):}\; r_i'=\frac{e_i}{\hat\sigma\sqrt{1-h_{ii}}}$$ $$\text{Studentized Deleted (external):}\; r_i^*=\frac{y_i-\hat y_i^{(i)}}{\hat\sigma_{(i)}\sqrt{1-h_{ii}}}=r_i'\sqrt{\frac{n-p-1}{n-p-r_i'^2}}\sim t_{n-p-1}$$

5.3 Cook's Distance

$$d_i=\frac{(\hat\beta^{(i)}-\hat\beta)^\top(X^\top X)(\hat\beta^{(i)}-\hat\beta)}{p\,\hat\sigma^2}=\frac{r_i'^2}{p}\cdot\frac{h_{ii}}{1-h_{ii}}$$
  • 综合 inconsistency($r_i'^2$)+ leverage($h_{ii}$)→ influence
  • 阈值:$d_i>1$ 严重;Tutorial 常用 $d_i>0.1$ 做筛选

5.4 三个概念必分清 ★

概念英文度量解释
高杠杆high leverage$h_{ii}$X 空间位置极端(x 远离均值)
不一致/异常outlier / inconsistency$|r_i^*|$y 偏离模型预测
高影响high influence$d_i$删除后对 $\hat\beta$ 变化大

5.5 系统诊断 Systematic Diagnostics(全局)

  1. plot(yhat, r) → homogeneity(漏斗形 → 异方差;曲线形 → 非线性)
  2. qqnorm(rsta) → normality(两端偏离 → 重尾/偏态)
  3. plot(x_j, r) → 该 $x_j$ 是否需非线性项 / 是否有遗漏变量
  4. plot(time, r) → 独立性

5.6 局部诊断 Local Diagnostics(个体)

  1. QQ plot of $h_{ii}$、$|r_i^*|$、$d_i$ 各自找极值
  2. 阈值筛查
  3. Formal outlier test:造 dummy $u_i$(第 i 个观测 = 1),拟合 lm(y~X+u_i),检验 $u_i$ 系数
  4. 同时检验 $m$ 个点用 Bonferroni: $t_{n-p-m}(\alpha/(2m))$

5.7 Surgical Unit 例 Summary(讲义标杆例)

n=54,4 预测 (X1 clot, X2 prog, X3 enzyme, X4 liver),响应 log-survival。

  • 按 $h_{ii}$ 标出:13,17,18,28,32,38,42,43,50,52
  • 按 $|r^*|$:37,9,27,38
  • 按 $d_i$:22,28,37,38
  • Bonferroni 联合检验后,除 obs 37 外都显著

5.8 R 代码(通用)

fit  <- glm(y ~ X, data=d)            # 或 lm()
yhat <- fitted(fit)
r    <- residuals(fit, type="pearson")
h    <- hatvalues(fit)
rsta <- rstandard(fit)                 # internally studentized r_i'
rstu <- rstudent(fit)                  # externally studentized r_i*
cook <- cooks.distance(fit)

par(mfrow=c(2,2))
plot(yhat, r);  qqnorm(rsta);  qqnorm(h);  qqnorm(rstu);  qqnorm(cook)

# 阈值筛
h[h > 2*p/n];  rstu[abs(rstu) > 2*sd(rstu)];  cook[cook > 0.1]

# Bonferroni simultaneous outlier test
qt(alpha/(2*m), n-p-m, lower.tail=FALSE)

6. L8 Remedial Measures 补救措施

6.1 四类问题四类 remedy

问题补救
Nonlinearity 非线性多项式/log/inverse/piecewise (从 L5)
Non-constant variance 异方差WLS / 方差稳定变换 VST / Box-Cox
Multi-collinearity 共线删变量 / Ridge / PCR
Non-normality 非正态Box-Cox / 变换

6.2 WLS 加权最小二乘 ★

$$\hat\beta_W = (X^\top W X)^{-1} X^\top W y,\quad W=\mathrm{diag}(w_i),\; w_i=1/\sigma_i^2$$ $$\mathrm{Var}(\hat\beta_W) = \sigma^2 (X^\top W X)^{-1}$$ 目标:$\min\sum w_i(y_i-x_i^\top\beta)^2$

6.3 估计权重的方法 Weight Estimation

方法 A(无重复,讲义主方法):$\log|r_i|$ 对 $\log\hat y_i$ 回归 → $\log|r_i|=\alpha_0+\alpha_1\log\hat y_i$,则

$$w_i = \hat y_i^{-2\hat\alpha_1}$$

方法 B:$|r_i|$ 对 $\hat y_i$ 线性回归 → $w_i = 1/\hat{|r_i|}^2$

方法 C(有重复):每组 $n_i$ 个观测,组内样本方差 $s_i^2$ → $w_i = n_i/s_i^2$

诊断加权模型:画 $\sqrt{w_i}\,r_i$ vs $\sqrt{w_i}\,\hat y_i$,应该 无明显 pattern(null pattern)。
陷阱:$R^2$ 不能用来比较 WLS 与 OLS。

6.4 VIF ★核心

★ 方差膨胀因子 $$\text{VIF}_j=\frac{1}{1-R_j^2}$$ $R_j^2$ = auxiliary regression ($x_j$ 对其他 X 的回归) $R^2$

等价形式(从 lm 输出手算):

$$\text{VIF}_j=\frac{\mathrm{Var}(\hat\beta_j)\cdot\sum_i(x_{ji}-\bar x_j)^2}{\hat\sigma^2}=\frac{SE(\hat\beta_j)^2\,\sum(x_{ji}-\bar x_j)^2}{\hat\sigma^2}$$

阈值:VIF > 5(严格)或 > 10(宽松)→ 严重共线。

两预测时 VIF 相等:$R_2^2=R_3^2=r_{23}^2$(对称),故 $\mathrm{VIF}_2=\mathrm{VIF}_3=1/(1-r^2)$。

6.5 Ridge Regression 岭回归

$$\hat\beta_{\text{ridge}}=(X^\top X+\lambda I)^{-1}X^\top y$$ $$\min\|y-X\beta\|^2+\lambda\|\beta\|^2$$
  • 有偏($\lambda>0$)但方差更小
  • $\lambda$ 越大 → coef 越靠 0(不为 0)
  • $\lambda$ 由 GCV 选(GCV 最小)
  • X 需标准化
library(MASS)
r <- lm.ridge(y ~ ., data=d, lambda=seq(0,50,0.1))
plot(r$lambda, r$GCV, type="l")
lambda.best <- r$lambda[which.min(r$GCV)]

6.6 PCR 主成分回归

谱分解 $X^\top X = Q\Delta Q^\top$($\tau_j$ = 特征值);主成分 $Z=XQ$:

$$\hat\beta_{\text{pcr}}=\sum_{j=1}^{k}\frac{q_j q_j^\top y}{\tau_j},\quad k\le p$$
  • $k=p$ 时 = OLS
  • $k<p$ 时:丢弃小特征值(共线方向)→ 方差降、偏差升
  • 与 Ridge 对比:Ridge 是连续 shrinkage,PCR 是硬截断
library(pls)
pcr.fit <- pcr(y ~ ., data=d, scale=TRUE, validation="CV")
validationplot(pcr.fit, val.type="MSEP")

6.7 方差稳定变换 Variance Stabilizing Transformation

一般原理:若 $\sigma(\mu)$ 已知,找 $g$ 使 $g'(\mu)\propto 1/\sigma(\mu)$。

响应类型$\sigma^2\propto$变换
比例 Proportion $\in[0,1]$$\mu(1-\mu)$$\arcsin\sqrt Y$
计数 Count$\mu$$\sqrt Y$
右偏 Right-skewed$\mu^2$$\log Y$
严重右偏$\mu^4$$1/Y$

6.8 Box-Cox Transformation ★

$$g_\lambda(Y)=\begin{cases}(Y^\lambda-1)/\lambda, & \lambda\ne 0\\ \log Y, & \lambda=0\end{cases}$$ 若 $\sigma\propto\mu^\alpha$,则最佳 $\lambda=1-\alpha$。

$\alpha\to\lambda$ 对应表(讲义考点):

$\alpha$323/211/20-1/2-1
$\lambda=1-\alpha$-2-1-1/20 (log)1/2 ($\sqrt{}$)1 (none)3/22

6.9 α 的估计

  • Method I:$\log s_i$ 对 $\log\bar y_i$ 回归(有重复时) / $\log|r_i|$ 对 $\log\hat y_i$(无重复时)。斜率 $=\hat\alpha$。
  • Method II:最小化 $\max_i(s_i/\bar y_i^\alpha)/\min_i(s_i/\bar y_i^\alpha)$(在候选 α 中网格搜索)。

6.10 R 实现

# WLS
fit = lm(y ~ X)
wreg = lm(log(abs(resid(fit))) ~ log(fitted(fit)))
w = fitted(fit)^(-2*coef(wreg)[2])
fit.w = lm(y ~ X, weights = w)

# Box-Cox
library(MASS); bc = boxcox(lm(y~X));  lambda = bc$x[which.max(bc$y)]

# VST
lm(asin(sqrt(y)) ~ X);  lm(sqrt(y) ~ X);  lm(log(y) ~ X)

7. L9 Generalized Linear Models 广义线性模型 重点

7.1 GLM 三要素 Three Components

  1. Random(随机分量): $y_i$ 来自某个指数族分布(Normal / Binomial / Poisson)
  2. Systematic(系统分量): 线性预测子 $\eta_i = \beta_0 + \sum\beta_j x_{ij}$
  3. Link(链接函数): $g(\mu_i) = \eta_i$,即 $\mu_i = g^{-1}(\eta_i)$
模型分布Link $g(\mu)$$\mu$ 形式Var
Linear regNormalidentity$\mu=\eta$$\sigma^2$
LogisticBernoullilogit $\log\frac{\mu}{1-\mu}$$\mu=\dfrac{e^\eta}{1+e^\eta}$$\mu(1-\mu)$
PoissonPoissonlog$\mu=e^\eta$$\mu$ (= mean)

7.2 Logistic Regression 逻辑斯蒂回归 ★

★ 核心公式 $$\log\frac{\mu(x)}{1-\mu(x)}=\beta_0+\beta_1 x,\qquad \mu(x)=\frac{e^{\beta_0+\beta_1 x}}{1+e^{\beta_0+\beta_1 x}}$$

Odds 几率: $O(x)=\mu(x)/[1-\mu(x)] = e^{\beta_0+\beta_1 x}$

Odds ratio 几率比(x 每增 1 单位):

$$\text{OR}=\frac{O(x+1)}{O(x)}=e^{\beta_1}$$

7.3 Logistic 推断 Inference

  • Wald z-test(大样本,非 t!):$z=\hat\beta_j/SE(\hat\beta_j)\sim N(0,1)$
  • CI for $\beta_1$: $\hat\beta_1\pm z_{\alpha/2}\,SE(\hat\beta_1)$
  • CI for odds ratio: $\exp(\text{CI for }\beta_1)$
  • Adjusted OR: 多协变量时 $e^{\beta_j}$ = 固定其他变量时 $x_j$ 每增 1 的 OR

7.4 其他 Binomial Links(备用)

  • Probit: $g(\mu)=\Phi^{-1}(\mu)$
  • Cloglog: $g(\mu)=\log(-\log(1-\mu))$
  • Cauchit: $g(\mu)=\tan(\pi(\mu-0.5))$

7.5 Poisson Regression 泊松回归 ★

$$P(Y=y)=\frac{e^{-\mu}\mu^y}{y!},\quad E[Y]=\mathrm{Var}[Y]=\mu$$ $$\log\mu=\beta_0+\beta_1 x,\qquad \mu(x)=e^{\beta_0+\beta_1 x}$$

Multiplicative change(x 增 1):

$$\frac{\mu(x+1)}{\mu(x)}=e^{\beta_1},\qquad \text{% change}=(e^{\beta_1}-1)\times 100\%$$

7.6 Offset 偏置项(Poisson 暴露量)

若每观测有不同"暴露" $m_i$(如人数、时间长度),用 rate $=\mu_i/m_i$:

$$\log\mu_i = \log m_i + \beta_0 + \sum_j\beta_j x_{ij}$$ $\log m_i$ 为 offset,系数固定为 1。

7.7 R 实现

# Logistic
fit <- glm(y ~ x1 + x2, family = binomial(link="logit"), data=d)
summary(fit); exp(coef(fit)); exp(confint(fit))   # OR and CI
predict(fit, newdata=new, type="response")         # predicted prob

# Poisson
fit <- glm(y ~ x1 + x2, family = poisson(link="log"), data=d)
# with offset:
glm(y ~ x1 + x2 + offset(log(exposure)), family=poisson, data=d)

# Model selection
library(MASS); stepAIC(fit)

8.4 Var-Cov 矩阵使用

对 contrast $L\hat\beta$: $\mathrm{Var}(L\hat\beta)=L\,\Sigma\,L^\top$,$SE=\sqrt{\mathrm{Var}}$。

9. 高频公式速查卡 Formula Sheet ★★★

I. 基础分布
$\hat\beta\sim N(\beta,\,\sigma^2(X^\top X)^{-1})$;  $\dfrac{\hat\beta_j-\beta_j}{SE(\hat\beta_j)}\sim t_{n-p}$;  $\dfrac{(n-p)\hat\sigma^2}{\sigma^2}\sim\chi^2_{n-p}$
II. Multiple Comparisons
Bonferroni: $t_{n-p}(\alpha/(2k))$
Scheffé: $\sqrt{(a-1)F_{a-1,\,n-a}(\alpha)}$
Tukey: $q_{a,\,n-a}(\alpha)/\sqrt 2$
Dunnett: 表 $d_{a,\nu,\alpha}$;unequal n 因子 $1+0.07(1-n_j/n_1)$
III. General Linear F-test
$F=\dfrac{(SSE_R-SSE_F)/q}{SSE_F/(n-p_F)}\sim F_{q,\,n-p_F}$
IV. Model Selection
$R_a^2 = R^2 - \frac{p}{n-p-1}(1-R^2)$
$C_p = SSE_M/\hat\sigma^2_{\text{full}} - n + 2|M|$
AIC $= n\ln\hat\sigma^2_M + 2j_M$; BIC $= n\ln\hat\sigma^2_M + j_M\ln n$
V. Diagnostics
$h_{ii}>2p/n$ → high leverage
$r_i' = e_i/[\hat\sigma\sqrt{1-h_{ii}}]$;  $r_i^* = r_i'\sqrt{(n-p-1)/(n-p-r_i'^2)}\sim t_{n-p-1}$
$d_i = \frac{r_i'^2}{p}\cdot\frac{h_{ii}}{1-h_{ii}}$
VI. WLS & Multicollinearity
$\hat\beta_W = (X^\top W X)^{-1}X^\top W y$, $w_i=1/\sigma_i^2$
$\text{VIF}_j = 1/(1-R_j^2) = SE(\hat\beta_j)^2\sum(x_{ji}-\bar x_j)^2/\hat\sigma^2$
Ridge: $(X^\top X+\lambda I)^{-1}X^\top y$
LASSO: $\min\tfrac{1}{2n}\|y-X\beta\|^2+\lambda\|\beta\|_1$
VII. Box-Cox
$g_\lambda(Y)=(Y^\lambda-1)/\lambda$ ($\lambda\ne 0$), $\log Y$ ($\lambda=0$)
若 $\sigma\propto\mu^\alpha$: $\lambda=1-\alpha$
VIII. GLM
Logistic: $\log\frac{\mu}{1-\mu}=\beta_0+\sum\beta_j x_j$;  OR $=e^{\beta_j}$
Poisson: $\log\mu=\beta_0+\sum\beta_j x_j$;  % change $=(e^{\beta_j}-1)\times 100\%$
Offset: $\log\mu_i=\log m_i+\eta_i$
Wald: $z=\hat\beta/SE\sim N(0,1)$
